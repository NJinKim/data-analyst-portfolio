1: 월간 보고서 작성

프로젝트 개요

매월초 투자사에 비즈니스 현황을 보고하는 PDF 파일 구성과 연구소에서 발간하는 동향 리포트를 데이터 마트로 구성하여 워드 및 파워포인트 자동화를 통해 문서를 생성하는 시스템을 구축했습니다.

문서 자동화

pptx win32com xlwings

데이터 처리

Pandas NumPy SQL

자동화 도구

Apache Airflow

데이터 마트 구성

  • 월간 보고서 데이터: 매월 비즈니스 현황 지표 통합
  • 동향 리포트 데이터: 연구소 발간 자료 자동 수집 및 정리

문서 자동화 시스템

  • 워드 문서 생성: 템플릿 기반 월간 보고서 자동 생성
  • 파워포인트 제작: 차트 및 그래프 포함 프레젠테이션 자동화
  • PDF 변환: 최종 문서 PDF 형태로 자동 변환

  • 데이터 파이프라인: 정기적 데이터 수집 및 처리 자동화
